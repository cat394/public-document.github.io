:::tip[ご存じでしたか？]
Web サイトを作成するときに重要な要素である HTML と CSS という技術があり、これだけでも Web サイトを作成することができます。<br />
しかしそれだけでは複数ページの場合、後で修正するのに非常に手間がかかってしまったり、リッチな訪問者体験を実現することが困難です。<br />
そのため、[Svelte](https://svelte.jp/)のような*JavaScript フレームワーク*と呼ばれるツールを使用することが重要なのです。
:::

## 終わりに

いかがでしょうか。これらの技術は現代において最も強力で効率的な Web サイトの制作方法です。<br />
これらを用いることで、Web サイトを作成する際の手間となる作業が大幅に省け、必要な作業に集中できるため高品質になります。

ここからは余談になりますが、こちらの Web サイトは SvelteKit ではなく、Astro という Web フレームワークの[starlight](https://starlight.astro.build/ja/) というドキュメントサイトを容易に作成するためのツールを使用しています。<br />
このように、私はよりより Web サイト作りができるように絶えず学習を続けており、上記の技術を採用した開発は現代において最も良い選択です。
