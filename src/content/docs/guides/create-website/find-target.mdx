## 目的による Web サイトの違い

Web サイトを作成する際に最初に行うべきは、**その目的を決める**ことです。<br />
多くの場合、自己紹介や店舗の宣伝が目的かもしれませんが、ここでの「目的」とはさらに深い意味を持ちます。<br />
それは、Web サイトを訪れる訪問者にどのようなアクションを取ってもらいたいか、どの情報を前面に出すべきかを明確にすることです。

例えば、あなたが自身のプロフィールを Web サイトで公開したいと考えたとします。<br />
その際、2 つのアプローチが考えられます。<br />
一つ目は、自分の趣味に焦点を当てたプライベートな Web サイトです。<br />
趣味が旅行やランニングなどのアウトドアに関することであれば、その情熱を表現するために、山や海などの自然を連想させる色彩やデザインを取り入れるかもしれません。

二つ目は、自分の専門性や職業的なキャリアを強調したパブリックな Web サイトです。<br />
この場合は、職務経歴やプロジェクトの詳細など、よりフォーマルな内容が中心になります。デザインもシンプルかつプロフェッショナル、あるいは業界に応じた特色を盛り込むことで、異なるメインメッセージを伝えることができます。

このように、Web サイトを通じて訪問者にどのようなイメージや行動を促したいかを明確にすることで、設計やコンテンツの方向性を決定します。さらに、訪問者が最終的に取る行動（例えば製品の購入、サービスの予約、情報のリクエストなど）を目的とすることで、より成果を得られる設計を行うことができます。目的が明確であればあるほど、訪問者にとって価値のある、効果的な Web サイトを構築することに繋がります。
