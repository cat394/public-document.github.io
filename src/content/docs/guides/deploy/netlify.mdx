## ガイド

Web サイトを公開することは、あなたのプロジェクトを世界に示すエキサイティングなステップです。Netlify を使えば、この手順は驚くほど簡単です。ここでは、メールアドレスで Netlify にサインアップし、ファイルをアップロードしてサイトを公開する手順をご紹介します。
ステップが少し長く感じたかもしれません。しかしここで行っていることは単純であり、Netlify でアカウントを作成し、ファイルを提出しているだけです。

### 1. Netlify にサインアップする

まずは、Netlify にサインアップすることから始めましょう。

1. **[Netlify](https://www.netlify.com/)にアクセスします。**

2. **「Sign up」をクリックします。**
   画面右上の「Sign up」ボタンを見つけてクリックします。

3. **メールアドレスでのサインアップを選択します。**
   いくつかのサインアップオプションが表示されますが、メールアドレスを使ったサインアップを選んでください。

4. **必要な情報を入力します。**
   メールアドレス、パスワード、その他必要な情報を入力します。

5. **サインアップを完了します。**
   入力した情報を確認し、「Sign up」ボタンをクリックしてサインアップを完了します。

### 2. Web サイトをアップロードする

サインアップが完了したら、次は Web サイトのファイルをアップロードします。

1. **Netlify ダッシュボードにログインします。**
   サインアップ後、Netlify のダッシュボードに自動的にログインされます。そうでない場合は、ログイン画面から手動でログインしてください。

2. **新しいサイトを作成します。**
   ダッシュボード上で「New site from Git」ではなく、「Sites」タブから「New site from upload」というセクションを探し、そこをクリックします。

3. **Web サイトのファイルをドラッグ＆ドロップします。**
   ビルド済みのサイトのフォルダを Netlify ダッシュボードにドラッグして、ドロップエリアに放ります。

4. **アップロードを待ちます。**
   ファイルのアップロードが完了するまでしばらくお待ちください。これには数分かかることがあります。

5. **公開を確認します。**
   アップロードが終わると、Netlify は自動的にサイトを公開します。公開されたサイトの URL が画面に表示されるので、クリックして実際の Web サイトを確認してください。

これであなたの Web サイトは無事に公開されました、おめでとうございます 🎉<br />
以上で、Netlify を使って Web サイトを公開するプロセスは完了です。あなたのサイトがオンラインでどのように表示されるかを確認し、必要に応じてさらなる更新を行ってください。このガイドが Web サイト公開のお役に立てれば幸いです。
